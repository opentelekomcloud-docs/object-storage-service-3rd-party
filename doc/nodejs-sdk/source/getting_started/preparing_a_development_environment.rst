Preparing a Development Environment
===================================

-  Download the recommended version from the `Node.js's official website <https://nodejs.org/en/>`__ and install it.
-  Download the latest version of Eclipse IDE for JavaScript and Web Developer from `Eclipse's official website <https://www.eclipse.org/downloads/packages/>`__ and install it.
