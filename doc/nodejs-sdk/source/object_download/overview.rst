Overview
========

OBS Node.js SDK provides abundant APIs for downloading objects in the following modes:

-  :ref:`Downloading an Object - Text-Based <obs_29_0502>`
-  :ref:`Downloading an Object - Streaming <obs_29_0503>`
-  :ref:`Downloading an Object - File-Based <obs_29_0504>`
-  :ref:`Downloading an Object - Range-Based <obs_29_0505>`
-  :ref:`Downloading an Object - Resumable <obs_29_0510>`

You can call **ObsClient.getObject** to download an object.
